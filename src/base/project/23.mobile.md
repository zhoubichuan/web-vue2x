# 六.其他功能（移动端）

::: tip 前言
主要介绍移动端打包
:::

::: demo

```vue
<template>
  <el-form>
    <h3 class="login-title">csvtojson</h3>
    <el-form-item>
      <el-input
        type="text"
        placeholder="请输入"
        v-model="task"
      />
    </el-form-item>
    <el-form-item>
      <web-button type="primary" @click="onSubmit()">
        转 换
      </web-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  name: "Login",
  data() {
    return {
        task: "/usr/src/zhoubichuan/prod/web-nginx/config/aaa.csv",
    }
  },
  methods: {
    async onSubmit() {
      const res = await this.$api.example.csvtojson(this.task)
      if(res.code === 200){
        this.$message.success("成功!")
      }
    },
  },
}
</script>

<style>

}

.login-title {
  text-align: center;
  margin: 0 auto 40px auto;
  color: #303133;
}
</style>
```
